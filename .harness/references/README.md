# 参考清单

> **TL;DR**: 仓库内置规范清单合集。Orchestrator 在 Phase 5-7 或专项审查时按需查阅。

所有检查清单已合并到 `checklists.md`，按章节组织：

| 章节 | 用途 | 查阅时机 |
|------|------|----------|
| 安全清单 | OWASP Top 10、输入校验、认证鉴权、注入防护 | Phase 5 安全审查 |
| 性能清单 | 后端/前端性能、Core Web Vitals | Phase 5 性能审查 |
| 测试清单 | JUnit 规范、Mockito 规则、反模式 | Phase 6/7 测试编写与审查 |
| 可访问性清单 | 键盘、ARIA、对比度、屏幕阅读器 | 前端无障碍审查 |

## 使用方式

Orchestrator 按需查阅 `checklists.md` 中对应章节。不需要预加载。
