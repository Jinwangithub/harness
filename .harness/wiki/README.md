# 项目知识库

## 用途

L3 按需查询层 — 存储业务文档、领域知识、架构说明等 Agent 按需查阅的内容。

## 推荐内容

- 业务领域概念说明
- 核心业务流程文档
- 上下游系统集成说明
- 部署架构图
- 数据库 ER 图
- API 接口文档
- 环境信息（开发/测试/生产）
- 第三方服务对接说明

## 使用方式

Agent 在遇到不清楚的业务概念时，主动查阅此目录。

```markdown
# 示例: 业务域文档

# 订单域

## 核心实体
- Order: 订单主表
- OrderItem: 订单明细
- OrderLogistics: 物流信息

## 状态机
待支付 → 已支付 → 已发货 → 已签收 → 已完成

```
